# PineScript-Builder

PineScript-Builder is a controlled, audit-first pipeline for ingesting TradingView Pine Script documentation and building a reliable foundation for downstream retrieval and strategy generation. The project prioritizes correctness, clarity, and long-term maintainability over speed.

## Core principles
- Correctness and reproducibility over convenience
- Auditability and explicit change control
- Version safety with mandatory Pine version metadata
- Agent governance with strict scope discipline

## Project Phases
- Phase 0 — Environment & Governance
- Phase 1 — Documentation Ingestion
- Phase 2 — Indexing & Retrieval (RAG)
- Phase 3 — Strategy Specification & Generation
